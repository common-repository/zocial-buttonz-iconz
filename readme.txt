100 % CCS3 Buttons and icons for social networks and others

== Description ==

The idea behind this project was to produce a consistent set of buttons that could be used for the range of social actions frequently taken in Web applications. These actions are often important goals for users, such as connecting third-party accounts or sharing content to third-party platforms, so their appearance has to be attractive and clear


== Installation ==

1. Unpack the plugin into `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. use shortcodes to bring it to life

some shortcodes

[zocialiconz class="zocial facebook"]Sign in with Facebook[/zocialiconz]
[zocialiconz class="zocial googleplus"]Sign in with Google+[/zocialiconz]
[zocialiconz class="zocial twitter"]Sign in with Twitter[/zocialiconz]

more shortcodes : http://www.ahmedag.com/blog/zocial-buttonz-iconz-wordpress-plugin/
